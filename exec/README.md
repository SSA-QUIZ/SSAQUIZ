# README



## 1. 프로젝트 세팅 방법

### 1. frontend

1. Clone repo

   ```bash
   $ git clone https://lab.ssafy.com/s04-final/s04p31a304.git
   ```

   

2. Install NPM packages

   ```bash
   $ cd frontend
   $ yarn install
   ```

   

3. run Client

   ```bash
   $ yarn serve
   ```

   

서버에 배포를 하고 싶으면 `/frontend` 경로에서 `yarn build` 한 후 나오는 `dist` 폴더를 사용하면 된다.
